# TeachTrack
SLO Evaluation platform for university courses.

TeachTrack helps UTA evaluate how well students are meeting Student Learning Outcomes (SLOs). Students complete structured surveys, and the platform generates dashboards for professors and course coordinators showing which SLOs are being met, which need attention, and how performance compares across sections.

## Tech Stack
- **Frontend:** React (Vite) + React Router + Tailwind CSS
- **Backeend:** Node.js + Express
- **Database:** PostgreSQL with raw SQL (pg)
- **Language:** Typescript

## Getting Started
1. **Clone the repo**
   ```bash
   git clone https://github.com/TeachTrackUTA/TeachTrack.git
   cd teachtrack
   ```
2. **Install Dependencies**
   ```bash
   npm install
   ```
3. **Start Development**
   ```bash
   npm run dev
